# NodeJS Blog
NodeJS Blog - CICCC WMAD Project


## How to run
- To generate sample data:

npm run db:seed

- To start the app at default Port 5000:

npm start

- Open the blog in your browser at:

http://localhost:5000/blog


## Tech Stack
- Node.js (backend)
- Express
- EJS
- Mongoose
- MongoDB
- CSS, Bootstrap, JavaScript (frontend)
